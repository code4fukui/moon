# moon

このリポジトリは、Nikon D90で撮影された2021年の中秋の名月（Harvest Moon）のオープンデータを提供します。データは、処理済みのJPEG形式とオリジナルのRAW（`.NEF`）形式の両方で利用可能です。

![2021年の中秋の名月](moon-400m-20210921.jpg)

*Photo by [@taisukef](https://twitter.com/taisukef/status/1440307372480626696)*

## データ

このリポジトリでは以下のファイルが利用可能です：

| ファイル | フォーマット | 説明 |
| --- | --- | --- |
| `moon-400m-20210921.jpg` | JPEG | 処理済み画像 |
| `DSC_1740.NEF` | Nikon D90 RAW | オリジナルのRAW画像データ |
| `DSC_1777.NEF` | Nikon D90 RAW | オリジナルのRAW画像データ |
